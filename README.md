### Installing

**Cloning the Repository**

```
$ git clone https://github.com/emanuelcorrea/vida-leve-mobile.git

$ cd vida-leve-mobile
```

### Running

After to downloaded or clonned the repositore, you can now run the app:

Android

```
$ flutter run
```
